## server side rendering
- Vue.js 是构建客户端应用程序的框架。默认情况下，可以在浏览器中输出 Vue 组件，进行生成 DOM 和操作 DOM。然而，也可以将同一个组件渲染为服务器端的 HTML 字符串，将它们直接发送到浏览器，最后将这些静态标记"激活"为客户端上完全可交互的应用程序。
> 把客户端的代码传递到服务端,由服务端回传字符串,再做渲染  可以认定为同构(就相当于2进制转换为10进制,都是一样的东西)
- 优点: 利于seo(search engine optimization) 搜索引擎最优化 
> seo抓取工具(google和bing只能对同步JavaScript进行抓取),如果初始化页面是菊花图,还需要等ajax异步返回数据,那么就需要ssr了
